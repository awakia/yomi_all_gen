# yomi_all_gen
ローマ字の読みの候補を全て生成するRubyスクリプト

## How to use

```
p generate('ono')
p generate('ohno')
p generate('omi')
p generate('otto')
p generate('kohama')
p generate('kondo')
p generate('n')
p generate('nn')
p generate('nnn')
p generate('nnnn')
p generate('no')
p generate('hanyu')
p generate('u')
p generate('yu')
p generate('yuko')
```

に対し、以下の出力をします。

```
$ ruby yomi_all_gen.rb
["オンオ", "オンオウ", "オンオオ", "オノ", "オノウ", "オウンオ", "オウンオウ", "オウンオオ", "オウノ", "オウノウ", "オオンオ", "オオンオウ", "オオンオオ", "オオノ", "オオノウ"]
["オウンオ", "オウンオウ", "オウンオオ", "オウノ", "オウノウ", "オオンオ", "オオンオウ", "オオンオオ", "オオノ", "オオノウ"]
["オミ", "オウミ", "オオミ"]
["オット", "オットウ", "オウット", "オウットウ", "オオット", "オオットウ"]
["コハマ", "コウハマ", "コウアマ"]
["コンド", "コンドウ", "コウンド", "コウンドウ"]
["ン"]
["ンン", "ン"]
["ンンン", "ンン", "ッン"]
["ンンンン", "ンンン", "ンッン"]
["ンオ", "ンオウ", "ンオオ", "ノ", "ノウ"]
["ハンユ", "ハンユウ", "ハニュ", "ハニュウ"]
["ウ"]
["ユ", "ユウ"]
["ユコ", "ユコウ", "ユウコ", "ユウコウ"]
```

無駄なものが多く生成されますが、検索など、読みの候補が全て生成されていれば良い場合に使うことが出来ると思います。
必要な部分は好きに改善してください。
